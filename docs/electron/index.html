Electron SPA

Electron is built on Chromium and the navigation model is pretty much the exact same as any other web browser. Electron needs ROOT, as / in electron means root of drive; not root of http server. There is no http server. Not having to go to a remote web server means loading pages in Electron should be pretty quick. Electron has node and it lets you start Chrome. So you can build a Windows app, or an OSX app. Electron apps are normally SPA. Our example also in one page.

  1. Run the command to extract an Electron SPA App example:

    $ mbakeX -e

  2. From the /elect folder install node modules:

    $ yarn

  3. Compile .pug in /elect/app/www folder and .scss/.sass in /elect/app/www/assets, eg:

    $ mbake .
$ mbake -s .

  4. Run the app from the elect/app folder with the command:

    $ npm start

    The application should show up.

  5. Inspect /elect/app/www/assets/router/spa-router.js, here you can see code:
